예시:
```json
"audio/bgm/theme_92": {
  "fileName": "Theme_92.ogg",
  "bytes": 1237413,
  "Crc": 1943281306
}
```

1. `Crc` 값으로 끝나는 DAT 파일 찾기<br>
   CRC 값: `1943281306`<br>
   파일: `1745218274191323495_1943281306.dat`
2. 해당 파일의 앞 숫자를 이름으로 가진 파일 찾기<br>
   파일: `1745218274191323495`
3. 해당 파일의 파일 크기 확인<br>
   Bytes 값: `1237413`<br>
   파일 크기: 1237413 바이트
